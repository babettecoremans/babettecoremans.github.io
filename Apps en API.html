Apps en API

Applicaties, oftewel apps, zijn de software waaruit de toepassingen laag uit bestaat. Denk bijvoorbeeld aan spotify, buienradar en de NOS app. De communicatie tussen verschillende lagen en tussen de onderdelen in die lagen wordt mogelijk gemaakt door interfaces. Software-interfaces, ofwel Application Programming Interfaces (API), worden gebruikt voor de communicatie tussen verschillende software-onderdelen. APIâ€™s zijn de interface tussen de logische laag en de toepassingen laag. Een app bestaat meestal uit onderdelen die afkomstig zijn van een API, bijvoorbeeld knoppen en tekstvelden. De API zorgt ervoor dat alle apps op een vergelijkbare manier werken waardoor consistentie ontstaat in the gebruik van een apparaat. Daarnaast kan een API ook verbinding maken met het internet. Een app doet dit altijd via de API van een besturingssysteem.
