Apps en API

Applicaties, oftewel apps, zijn de software waaruit de toepassingen laag uit bestaat. Denk bijvoorbeeld aan spotify, buienradar en de NOS app.

De communicatie tussen verschillende lagen en tussen de onderdelen in die lagen wordt mogelijk gemaakt door interfaces. Software-interfaces, ofwel Application Programming Interfaces (API), worden gebruikt voor de communicatie tussen verschillende software-onderdelen. Meestal bestaat een app uit onderdelen van een API, bijvoorbeeld knoppen en tekstvelden. Door de ontstaat consistentie bij het gebruik. Dit betekent dat alle apps op een vergelijkbare manier werken. Daarnaast kan een API ook verbinding maken met het internet. Zo haalt een informatieve app de benodigde data uit een database via de web-API.
